### A. 结构化知识框架 (Module 2: Introduction to CS7637)

**2.0 课程目标与成果**

- **学习目标 (Learning Goals)**:
    
    - 1. 学习KBAI的核心方法：包括知识表示方案、记忆组织方法、推理方法、学习方法、认知架构及元推理 (metareasoning) 。
        
    - 2. 学习KBAI处理的常见任务：如分类、理解、规划、解释、诊断和设计 。
        
    - 3. 学习AI智能体如何运用方法解决任务 。
        
    - 4. 学习KBAI与认知科学的关系：用人类认知理论启发AI设计，用AI技术对人类认知提出可检验的假设 。
        
- **学习成果 (Learning Outcomes)**:
    
    - 1. 能够设计、实现、评估和描述基于知识的AI 。
        
    - 2. 能够运用KBAI策略解决实际问题 。
        
    - 3. 能够利用AI智能体的设计来反思人类认知，反之亦然 。
        
- **学习策略 (Learning Strategies)**:
    
    - **通过示例学习 (Learning by Example)**: 每节课都以一个贯穿始终的例子开始 。
        
    - **通过实践学习 (Learning by Doing)**: 课后通过多部分练习来实践所学推理过程 。
        
    - **基于项目的学习 (Project-Based Learning)**: 课程围绕一系列有挑战性的项目构建 。
        
    - **个性化学习 (Personalized Learning)**: 学生可自选学习顺序和节奏，并获得个人反馈 。
        
    - **通过反思学习 (Learning by Reflection)**: 通过课后小测和项目设计报告进行反思 。
        

**2.1 计算心理测量学 (Computational Psychometrics)**

- **心理测量学 (Psychometrics)**: 研究人类智力、能力和知识的学科 。
    
- **计算心理测量学**: 设计能够参加人类智力、知识或能力测试的计算智能体 。
    
- **研究方法**: 比较智能体的表现和错误与人类的表现和错误，若两者相似，则可推测其推理过程可能与人类相似 。
    
- **本课程应用**: 设计能够解决瑞文推理测试 (Raven's Test) 的AI智能体 。
    
- **认知连接**:
    
    - 瑞文测试是一种纯视觉类比问题测试，不含文字 。
        
    - 自闭症患者在该测试上的表现与神经典型人群相当，但在其他含语言的智力测试上则不然 。
        
    - 这可能暗示自闭症患者的思维策略更适合视觉推理 。
        

**2.2 课程核心项目: 瑞文渐进矩阵测试 (Raven's Progressive Matrices)**

- **背景**:
    
    - 1930年代为测试一般人类智力而设计 。
        
    - 是最广泛使用、最可靠的智力测试之一，由60道多项选择视觉类比题组成 。
        
- **问题类型**:
    
    - **2x1 矩阵**: A is to B as C is to D。课程中用作构建AI的起点 。
        
    - **2x2 矩阵**: 不仅有水平关系 (A:B::C:D)，还有垂直关系 (A:C::B:D) 。
        
    - **3x3 矩阵**: 需同时考虑水平、垂直和对角线关系 。
        
- **关键挑战与洞察 (通过示例展示)**:
    
    - **生成与测试 (Generate and Test)**: 当一个假设不成立时，需要返回并生成新的假设来解决问题 。
        
    - **歧义性 (Ambiguity)**: 同一个问题可能有多种合理解释（如旋转vs.镜像），导致多个看似正确的答案 。
        
    - **推理策略多样性**: 解决不同问题需要根本不同的推理过程（如3x3矩阵中的XOR逻辑问题） 。
        
    - **AI与认知的双向启发**: 研究人类如何解决问题可以启发AI设计；反之，观察AI的解决过程也可以为人类认知提供假设 。
        
在瑞文测试的一个2x2矩阵示例中，讨论了除了旋转 (rotation) 和镜像 (reflection) 之外的第三种看待问题的方式，这种方式不依赖于语义转换，而是依赖于整体图像。这种方式被称为什么？

**“看哪个图像能补完整个画面” (which image completes the overall picture)** 。这是一种“纯视觉” (strictly visual) 的解题思路，而不是依赖于旋转或镜像这样的语义转换

**2.3 智力的定义**

- **问题**: 如果一个AI能通过智力测试，它是否就是智能的？。
    
- **观点**: 难以定义。类似于生命科学中对“生命”的定义，认知科学研究智力，但不一定能给出一个公认的定义 。
    
- **本课程观点**: **知识是人类水平智能的核心 (knowledge is central to human level intelligence)** 。
    

**2.4 CS7637的七大原则 (Principles of CS7637)**

- 1. 智能体使用
    
    **知识**（组织在知识结构中）来指导推理 。
    
- 2.
    
    **学习**通常是**增量**的 。
    
- 3.
    
    **推理**是**自顶向下**的（利用知识产生预期），而不仅仅是自底向上的 。
    
- 4. 智能体将
    
    **方法与任务相匹配** 。
    
- 5. 智能体使用
    
    **启发式方法 (heuristics)** 寻找“足够好”的解，而非最优解（有限理性） 。
    
- 6. 智能体利用问题中
    
    **重复出现的模式** 。
    
- 7.
    
    **推理、学习和记忆**相互约束、相互支持，课程将构建三者的统一理论 。
